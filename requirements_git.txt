# utils3d MUST be installed BEFORE MoGe (and listed first here). PyPI has an
# unrelated "utils3d" stub package at v0.1.3 that gets pulled in transitively
# and shadows the real one — its `.numpy` submodule is empty, so any call to
# utils3d.numpy.icosahedron / .intrinsics_from_fov / .extrinsics_look_at /
# .image_uv / .depth_edge crashes with AttributeError (see panorama_utils.py:37
# and gen_gs_data.py 147/340/342/417/472). Installing the real one from
# EasternJournalist first wins the version race; --no-deps below stops MoGe
# from clobbering it on its way in.
